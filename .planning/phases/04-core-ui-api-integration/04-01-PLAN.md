Add static file serving infrastructure to main.go. This integrates with the existing gRPC-Gateway HTTP server (created around line 275).

**1. Add embed imports at top of file:**
```go
import (
    "embed"
    "html/template"
    // ... existing imports
)
```

**2. Add embed.FS directives after imports (before const declaration):**
```go
//go:embed web/static
var staticFS embed.FS

//go:embed web/templates
var templatesFS embed.FS
```

**3. Modify newGRPCGatewayHTTPServer function (around line 349-370) to add static routes:**
Before the line `mux.Handle("/", handler)` add:

```go
// Serve static files
staticFiles, _ := fs.Sub(staticFS, "web/static")
mux.Handle("/static/", http.StripPrefix("/static/", http.FileServer(http.FS(staticFiles))))

// Serve tournaments page
mux.HandleFunc("/tournaments", func(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "text/html; charset=utf-8")
    tmplContent, err := templatesFS.ReadFile("web/templates/base.html")
    if err != nil {
        http.Error(w, "Template not found", http.StatusInternalServerError)
        return
    }
    tmpl, err := template.New("base").Parse(string(tmplContent))
    if err != nil {
        http.Error(w, "Template parse error", http.StatusInternalServerError)
        return
    }
    data := map[string]interface{}{
        "Content": "<h1>Tournaments</h1><p>Loading...</p>",
    }
    tmpl.Execute(w, data)
})
```

**4. Add fs import if not present:**
```go
import "io/fs"
```

**Implementation notes:**
- Place static routes BEFORE the catch-all `mux.Handle("/", handler)` to prevent gRPC-Gateway from intercepting
- Use http.StripPrefix to remove "/static/" from path before FileServer lookup
- Inline template parsing is fine for now (single template, no complexity)
- No caching headers yet - keep it simple
- Template data with placeholder content will be replaced by JavaScript in Plan 02
